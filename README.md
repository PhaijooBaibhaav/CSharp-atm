# Csharp-ATM
## This is a ATM program made with c#. You can create an account deposit money, withdraw money and also see balance. All of that data is stored in a file in the folder data.

## Requirements: 
- Dotnet SDK
- Git

# Installation
````bash
git clone https://github.com/PhaijooBaibhaav/CSharp-atm
````

 ````bash
cd CSharp-atm
````

````bash
dotnet run
````
